## 1. سياسة عدم الأسرار (No-Secrets Policy)

### ممنوع منعاً باتاً:
- ❌ رفع API tokens أو keys
- ❌ رفع connection strings
- ❌ رفع أي credentials

### مسموح:
- ✅ الأسرار في Cloudflare/Render/Vercel dashboards
- ✅ الأسرار في GitHub Secrets
- ✅ ملفات template فقط
